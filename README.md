# Instagram Clone
## Tech

Instagram Clone is build from these Technologies:

- [React Js] - For frontend!
- [Material UI] - for great custome components.
- [node.js] - For Runtime Enviroment
- [MongoDb] - For Database
- [Express] - For creating backend server
